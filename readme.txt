文件说明:
make.bat   批处理文件,用来链接编译文件执行整个应用的完成
makefile   开发人员编写的编译脚本
gmake.exe  编译程序,windows下GNU-MAKE工具
makerom.exe windows下GNU-MAKE的辅助工具,把几个临时的rom拼装成一个rom 
tag.txt    按规范填写的TAG格式文件,包含标签版本号,发布版本号,提交者,发布日期,主要改动,版本处理,适用BIOS程序,升级/生产说明.

目录说明:
BIN 	存放提交给客户的目标文件,库文件,参数文件等
DOC	操作手册,接口文档等等
SRC	源程序,所有的.C文件
INC	源程序的头文件.H
LIB	库文件,用于链接的.S库文件
LIBAPI	公共库的头文件
BIOSAPI	BIOS的API文件,其中包含底层提供的初始化程序链接库,API函数链接库以及所包含的头文件等
OBJ_ERR	这个目录是留着给设计过程中所得到的中间文件,.O文件和ERR文件
TOOL	该目录存放应用的辅助工具,可以为空.为了节约上传空间,建议不要把如下载器等通用的工具不要放在此目录中.
